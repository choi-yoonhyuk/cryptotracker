> 실시간으로 코인의 시세를 확인할 수 있는 'Coin' 입니다.
> </br>

수많은 코인 중 Top 100을 선정해 사용자에게 제공합니다. </br>
그래프를 통하여 복잡한 데이터를 조금 더 알아보기 쉽게 제공합니다.
다크모드와 라이트모드를 적용하여 사용자 환경에 맞게 사용할 수 있습니다.

---

## 설치 방법

```sh
npm install
```
